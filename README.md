# My Personal Website
 
A simple personal website built with plain HTML and CSS. No frameworks, no build tools.
 
## What's on it
 
- **Header** — welcome banner with a dark gradient background
- **About Me** — short intro
- **My Team (SigmaProg)** — cards for each team member with a GitHub link
- **My Favorite Games** — a short list with descriptions
- **Footer** — copyright notice
## Files
 
```
├── index.html   → page structure and content
└── style.css    → all styling (colors, layout, fonts, effects)
```
